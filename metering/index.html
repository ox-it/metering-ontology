Introduction

An ontology for modelling meters of various kinds and their readings.

Changes From Previous Version

    Namespace

    The URI for this vocabulary is 

    http://vocab.ox.ac.uk/meter#

    When used in XML documents the suggested prefix is meter

    Each class or property in the vocabulary has a URI constructed by appending a term name to the vocabulary URI. For example:

    http://vocab.ox.ac.uk/meter#unit
http://vocab.ox.ac.uk/meter#MeterableResource

    The term name for a class always starts with an uppercase character. Where the term name is comprised of multiple concatenated words, the leading character of each word will be an uppercase character. For example: 

    MeterableResource
Metric

    The term name for a property always starts with an lowercase character. Where the term name is comprised of multiple concatenated words, the leading character of the second and each subsequent word will be an uppercase character. For example: 

    unit
unitAbbrev

    Summary of Terms

    This vocabulary defines 4 classes and 2 properties .

    Term NameTypeDefinition
    CumulativeMetricclassA class for metrics measured relative to some other point in the past.
    InstantaneousMetricclassA class for metrics measured at a single point in time.
    MeterableResourceclassA physical phenomenon that can be quantified in one or more dimensions.
    MetricclassAn abstract class for metering metrics.
    unitpropertyThe unabbreviated name of the units used when recording a given metric.
    unitAbbrevpropertyThe abbreviated name of the units used when recording a given metric.
